# Desafio Bootcamp de Desenvolvimento Java

Este é um desafio proposto como parte do Bootcamp de Desenvolvimento Java da plataforma Digital Innovation One (DIO). O objetivo deste desafio é implementar um sistema simples de gerenciamento de Bootcamps, Cursos, Mentorias e Desenvolvedores em Java.

## Descrição do Projeto

O projeto consiste em uma aplicação Java que simula o gerenciamento de Bootcamps, Cursos, Mentorias e Desenvolvedores. O sistema permite:

- Criar Bootcamps com nome, descrição, data de início e término, e associar conteúdos como cursos e mentorias.
- Criar Cursos com título, descrição e carga horária.
- Criar Mentorias com título e descrição.
- Inscrever Desenvolvedores em Bootcamps e registrar sua progressão nos conteúdos.
- Calcular o XP total acumulado por cada Desenvolvedor com base nos conteúdos concluídos.

## Estrutura do Projeto

O projeto é composto por algumas classes principais:

- `Bootcamp`: Representa um bootcamp com nome, descrição, data de início e término, e uma lista de conteúdos.
- `Curso`: Representa um curso com título, descrição e carga horária.
- `Mentoria`: Representa uma mentoria com título e descrição.
- `Dev`: Representa um desenvolvedor com nome, e listas de conteúdos inscritos e concluídos.
- `Main`: Classe principal contendo o método `main` que demonstra o funcionamento básico do sistema.

## Como Executar o Projeto

Para executar o projeto, basta seguir os seguintes passos:

1. Clone este repositório para sua máquina local.
2. Abra o projeto em sua IDE Java favorita.
3. Execute a classe `Main`.
